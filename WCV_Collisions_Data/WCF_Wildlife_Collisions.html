WCV Collision Data Audit

Field Naming with Comments on Fields

The source spreadsheet field names had spaces in the names and were converted to upper camel case. Some derived variables were calculated to ease with further processing of the data. The variables begin with a D. In the table below, these names do not have Source Names.

Source Name Output Name Comment
Organization Name OrganizationName All values are the same.
Case Number CaseNumber No duplicate values.
Patient ID PatientID No duplicate values.
Common Species Name CommonSpeciesName Inconsistent capitalization. Also may be a problem with automation as new species are rescued.
NA DVertebrate To simplify added 4 vertebrate categories since there are over 100 common species. Need to confirm classification.
Date Admitted DateAdmitted No issues found.
NA DDateAdmittedYear Year is derived from DateAdmitted.
NA DDateAdmittedMonth Month is derived from DateAdmitted.
NA DDateAdmittedDOM Day of Month is derived from DateAdmitted.
NA DDateAdmittedDOW Day of Week is derived from DateAdmitted.
Circumstances of Rescue CircumstancesOfRescue All values are the same.
Rescue State RescueState All values are the same.
Rescue Juristiction RescueJurisdiction Jurisdiction is misspelled in the original spreadsheet.
Rescue Address RescueAddress No issues found.
Other Rescue Information OtherRescueInformation Many blank values. This is OK.
Latitude Latitude All valid values in Commonwealth of Virginia. 5 missing values. These records will not be mapped in a dashboard which may create an issue.
Longitude Longitude All valid values in Commonwealth of Virginia. 5 missing values. These records will not be mapped in a dashboard which may create an issue.
Elevation Elevation The range of elevation values is reasonable for the Commonwealth of Virginia. Did not evaluate each elevation.
Disposition Disposition No issues found.
